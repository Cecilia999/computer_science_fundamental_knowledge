# JAVA

### Java 语言的特点

- **面向对象**

  - 概念：类、对象
  - 特性：继承、封装、多态

- **跨平台 jvm**

  - 跨平台性：不同的操作系统通过安装不同的 java 虚拟机（JVM Java Virtual Machine），由 JVM 负责 java 程序在系统的运行

- **java 核心机制**

  - java 虚拟机（JVM）
  - 垃圾回收机制（GC）

- **JDK 与 JRE**

  - JDK(Java Development Kit  Java 开发工具包) 包含开发工具（能编译打包 java 文件）
  - JRE（Java Runtime Environment  Java 运行环境）能运行 java 文件
    _包含关系_   **JVM<JRE<JDK**
