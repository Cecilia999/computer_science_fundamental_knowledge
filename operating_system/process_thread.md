# Process VS Thread

### 1. 进程与线程之间有什么区别

- 进程：进程是程序的实体，指在系统中正在运行的一个应用程序；程序一旦运行就是进程；是系统进行资源分配和调度的基本单位，是操作系统结构的基础。
- 线程：是进程中的实际运作单位，程序执行的最小单位，一条线程指的是进程中一个单一顺序的控制流，一个进程中可以并发多个线程，每条线程并行执行不同的任务。
